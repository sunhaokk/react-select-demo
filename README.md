React-Select允许您通过使用components属性用您自己的默认组件替换来扩大布局和功能。这些组件具有所有当前的支持和状态，可让您实现梦想中的一切。

每个组件均会接收commonProps，这些属性会传播到该组件上。这些包括：

clearValue
getStyles
getValue
hasValue
isMulti
isRtl
options
selectOption
setValue
selectProps



isClearable 用来控制右侧的叉号显示不显示
isSearchable  用来控制是否可以搜索
